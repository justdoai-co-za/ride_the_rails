# Ride the Rails

Africa's Hidden Gravel Frontier - Mapping the untapped 36,000 km network of South Africa's railway service roads.

## About

Ride the Rails is a project to map and maintain South Africa's extensive network of railway service roads for bikepacking and gravel cycling.

## Website

This repository contains the landing page for ridetherails.co.za

Visit: [https://justdoai-co-za.github.io/ride_the_rails/](https://justdoai-co-za.github.io/ride_the_rails/)

## Project Features

- 36,000 km of railway service roads across all 9 provinces
- Community-maintained routes and custodians
- Interactive maps and GPX downloads
- Micro-donation system to support trail maintenance
